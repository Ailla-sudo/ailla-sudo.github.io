JavaScript基础

发表于 2021-10-12 分类于 Web语言

1. 在网页中使用JavaScript

1.1 Internal

首先将< script>标记插入到HTML页面的某个地方，然后在< script>标记对之间根据需要编写相关的JavaScript脚本，一般将< script>标记写在< head>< /head>之间或< body>< /body>之间。

1.2 External

通常将JavaScript代码编写为一个独立的以js为扩展名的脚本文件，以便重用JavaScript代码。

1.3 函数

为了让代码井井有条，应将代码要完成的每个“任务”包装到一个函数里。
function showMessage() {
……
}

可以给函数接收一个形参（形参的个数无限制）：
function showMessage(customMessage) {
alert(customMessage);
}

调用函数并提供实参：
showMessage(“Nobody likes an argument.”);

既可以接收信息，又可以返回信息：
function multiplyNumbers(numberA,numberB) {
return numberA*numberB;
}

1.4 响应事件

事件名 说明 适用元素
onClick 鼠标单击元素时触发 所有元素
onMouseOver 鼠标悬停在元素上时触发 所有元素
onMouseOut 鼠标从元素上移开时触发 所有元素
onKeyDown 按下某个键时触发 <select>、<input>、<textarea>、<a>、<botton>
onKeyUp 释放某个键时触发 <select>、<input>、<textarea>、<a>、<botton>
onFocus 控件接收到焦点时（也就是鼠标指针位于控件中，可以输入的时候）触发。这里所说的控件包括文本框、复选框等 <select>、<input>、<textarea>、<a>、<botton>
onBlur 焦点从控件移开时触发 <select>、<input>、<textarea>、<a>、<botton>
onChange 修改了控件中的值之后触发。对文本框而言，当移动到下一个控件时才会触发 <select>、<input type=”text”>、<textarea>
onSelect 选择输入控件中的部分文本时触发 <input type=”text”>、<textarea>
onError 浏览器下载图片失败时触发（通常是因为URL错误） <img>
onLoad 浏览器下载完新页面或加载完对象（如图片）时触发 <img>、<body>
onUnload 浏览器卸载页面时触发（在浏览器地址栏中输入新URL或点击链接时发生，而且是在浏览器加载新页面前发生） <body>

2. JavaScript语言基础

2.1 变量

用var关键字后跟变量名声明。如：var myMessage;

2.2 null值

如果变量值是null，那就表示给定对象不存在或是空的。

2.3 变量作用域

局部变量：在函数内部创建的变量，该变量在函数运行的时候存在。

全局变量：在函数外部创建变量，该变量会在浏览器加载页面期间始终存在。

2.4 变量数据类型

在JavaScript中，变量可以保存不同类型的数据，比如文本、整数、浮点数、数组和对象。但是，无论在变量中保存什么值，都是使用同一个var关键字。

2.5 运算

var myNumber=(10+5)*2/5; 这行代码会遵循标准的数学运算法则，结果为6。

var firstName=”Sarah”;
var lastName=”Smithers”;
var fullName=firstName+” “+lastName; 使用加号（+）运算符可以把多段文本拼接成一个长字符串。

2.6 条件逻辑

逻辑运算符：==、!=、===（精确等于(值和数据类型)）、!==（非精确等于）、!、<、>、<=、>=、&&（逻辑与）、||（逻辑或）

3. 与页面交互

3.1 操作元素

页面中的每个元素都是一个对象，可以通过JavaScript代码检测和修改。
JavaScript为寻找对象提供了简单的方法：document.getElementById()。这里的document对象表示的是整个HTML文档。

一些常用的HTML对象属性：

className用于取得和设置class属性

innerHTML用于读取或修改元素内部的HTML标记

parentElement保存着包含当前元素对应的HTML对象

style保存有所有CSS属性，决定着相应HTML元素的外观

tagName保存当前对象的HTML元素的标签名

一些常用的方法：

**getAttribute()**返回指定属性名的属性值

**setAttribute()**添加指定的属性并为其赋指定的值

insertChild()

**appendChild()**向节点添加最后一个子节点

removeChild()
